# Portfolio Website

This is a simple personal portfolio website built using **HTML** and **CSS**.  
The website consists of two pages: **Home** and **Contact**, designed with a clean and minimal layout.

The Home page introduces my profile and skills, while the Contact page provides an easy way to get in touch.  
This project focuses on practicing core front-end concepts without using any external frameworks or libraries.

### Technologies Used
- HTML5  
- CSS3  

### Pages
- **Home** – Overview of my profile and skills  
- **Contact** – Contact information / form  

### Features
- Simple and clean design  
- Responsive layout  
- Easy navigation between pages  

### Purpose
This project was created to strengthen my fundamentals in HTML and CSS and to build a basic personal portfolio website.
